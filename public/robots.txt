# FastQuote robots.txt
# Discoverability — Wave 1 (2026-06-30) + bug-hunt fix #4 (2026-06-30)
#
# We WANT to be crawled and cited. Reputable search and AI crawlers are
# explicitly allowed. Authenticated and non-public surfaces are disallowed.
#
# IMPORTANT (bug-hunt 2026-06-30 #4): per the robots.txt spec, a crawler
# that matches a named User-agent block IGNORES the wildcard `*` block
# entirely — only the named block's rules apply. So every named-bot
# section below carries the SAME Disallow stanza as the wildcard. If
# you add a new private path, add it to ALL the Disallow groups below
# (or move private paths under a single prefix like /private/* and add
# one rule per group).
#
# Consensus AI-crawler list sourced from the ai-robots-txt project.
